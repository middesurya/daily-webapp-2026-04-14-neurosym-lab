## Why This Idea

Neuro-symbolic AI is THE breakthrough story of April 2026:
- Researchers demonstrated 100× energy reduction while improving accuracy
- IBM-MIT Watson AI Lab is driving this research
- NVIDIA GTC 2026 featured neuro-symbolic approaches for embodied AI
- No interactive playground existed — we filled that gap

## Features Shipped

1. **Reasoning Arena** — 3 interactive challenges:
   - Visual Sudoku: CNN digit recognition + constraint propagation solver
   - Scene Graph: Object detection + first-order logic inference chains
   - Word Problems: NLP entity extraction + algebraic equation building

2. **Pipeline Builder** — Drag-and-drop composition of 10 neural & symbolic modules

3. **Energy Dashboard** — Real-time FLOPs/memory/latency comparison with CO₂ impact and D3.js scaling charts

4. **Knowledge Graph Explorer** — Force-directed graphs with 3 presets (Animals, Family, Physics) and symbolic inference rule propagation

## Tech Stack

- Single HTML file (69 KB)
- D3.js v7 for graphs and charts
- Pure JavaScript symbolic reasoning engines
- No build tools, no frameworks, no external data fetches
- Dark glassmorphism UI with neon accents

## Lessons Learned

- D3.js force simulations need careful parameter tuning for knowledge graphs
- Embedding all data inline keeps the app fast and reliable
- Constraint propagation is surprisingly elegant for visual puzzles
- The split-panel comparison UI effectively communicates the neural vs symbolic tradeoff
